# Lego dress up spēle

## Īss apraksts
Programmā ir iespējams apģērbt divus lego tēlus un ievadīt viņu vecumu un vārdu un tad apskatīt darba pieteikuma lapiņu.
Programmas tēma ir lego, tādēļ vairāki elementi ir ar lego tēmu ieskaitot mūzika.

## Programmas izstrādes process
Programma tika izstrādāta Unity izstrādes vidē un programma tika skriptēta izmantojot C#
Visi asseti tika atrasti internetā un daži papildus modificēti.
